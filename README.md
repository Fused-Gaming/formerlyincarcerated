# Bitcoin Land Bond - Criminal Asset Recovery Initiative

> **Deploying $15B in seized cryptocurrency to fund permanent deed-restricted housing for 600,000+ formerly incarcerated people annually.**

## 🚀 Quick Start

### Prerequisites
- Node.js 16+ installed
- npm installed
- GitHub account with Fused-Gaming organization access
- Vercel account (free at vercel.com)

### Local Development (5 minutes)
```bash
# Clone the repository
git clone https://github.com/Fused-Gaming/formerlyincarcerated
cd formerlyincarcerated

# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:3000
```

### Deploy to Vercel (2 minutes)
```bash
# Build for production
npm run build

# Deploy to Vercel
vercel --prod
```
